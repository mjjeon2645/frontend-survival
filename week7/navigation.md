# 4. Navigation

## 강의 요약 및 노트

- Link, NavLink
- Navigate -> 무조건 리다이렉션
- 리다이렉션을 위한 route 테스트가 죽어버리는 이유는 polyfill 문제. 예전에 `fetch`에 대해서도 폴리필 한 것 처럼 `whatwg-fetch`를 임포트해서 해결 가능

- Navigate를 사용해도 되고, useNavigate 훅으로 navigate를 불러오는 방법을 더 많이 쓰긴 함
