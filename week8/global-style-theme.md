# 6. Global Style & Theme

## 강의 요약 및 노트

- styled-reset을 설치하여 모든 속성값을 초기화시킴

```bash
npm i styled-reset
```

- 글로벌 스타일을 지정하는 것은 styled-components에 포함되어 있고 `createGlobalStyle`을 통해 만들 수 있음
